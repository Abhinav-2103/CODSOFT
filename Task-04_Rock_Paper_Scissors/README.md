# Task 4 - Rock-Paper-Scissors Game

## Objective:
Create an interactive Rock-Paper-Scissors game between a user and the computer.

## Features:
- User inputs choice: rock, paper, or scissors.
- Computer randomly selects a choice.
- Game logic determines the winner.
- Results are displayed with a score tracker.
- Option to play multiple rounds.

## Technologies Used:
- Python
- `random` module

## How to Run:
1. Open a terminal.
2. Navigate to the folder containing `rps_game.py`.
3. Run the game:
   ```bash
   python game.py
